Baixando:
  
  1 - Baixe o .zip do repositorio e descompacte
  
  3 - Baixar godot 4.5 ou 4.6
  
  4 - Importe o arquivo project.godot

Anexando uma nova provincia:

  1 - Procurar pelo arquivo da provincia na pasta "provinces"

  2 - Mudar o "owner = (País x)" para "owner = (País y)"
  
  obs (O nomes dos paises são definidos por tags, por exemplo: Colombia = COL, para descobrir a tag pesquise dentro do nó "contries_data")

Criando um País:

  1 - Criar o arquivo .txt do País na pasta "countries" com a cor desejada, formato: "color = { x x x }" (em caso de erro colocar mais 1 espaço entre as cores)

  2 - crie o arquivo .txt com o nome da Tag do País na pasta "countries_color" com tambem a mesma cor do outro arquivo
